# 📖 About

NiNSupply is a full-stack MERN application built to streamline supply chain and inventory management. The platform enables businesses to manage products, monitor inventory, organize suppliers, and improve operational efficiency through a secure, scalable, and responsive web application.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/najatul6/NiNSupply-Client.git
```

Move into the project

```bash
cd NiNSupply-Client
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
